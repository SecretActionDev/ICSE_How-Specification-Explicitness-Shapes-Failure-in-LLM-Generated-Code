Prompt 5: what does installing python with pip mean? It has some commands in bash, how do i execute them?
LLM Response 5: Great question—this is one of those phrases everyone throws around without explaining 😅
Let’s break it down cleanly.

---

## What “installing Python with pip” actually means

There are **two different things** involved:

### 1️⃣ Python

Python is the **programming language** itself.

You install it once on your system:

```bash
python --version
```

or

```bash
python3 --version
```

If that prints a version number → Python is installed.

---

### 2️⃣ pip

**pip** is Python’s **package manager**.
It installs **Python libraries** (like `numpy`, `requests`, `pandas`, etc.).

So when someone says:

> “Install this with pip”

They mean:

> “Use pip to download and install a Python package”

Example:

```bash
pip install numpy
```

You are **not installing Python itself** here—just an add-on library.

---

## How to execute pip / bash commands

Those commands go in a **terminal**, not inside Python or VS Code files.

### Option A: VS Code (recommended)

1. Open VS Code
2. Open your project folder
3. Open the terminal:

 * **View → Terminal**
 * or `Ctrl + `` (backtick)
4. Make sure it says **bash**, **zsh**, or **PowerShell** at the top
5. Paste the command and press **Enter**

Example:

```bash
pip install requests
```

---

### Option B: System terminal

* **Linux**: open Terminal
* **macOS**: open Terminal
* **Windows**:

 * Use **PowerShell**, or
 * Use **Git Bash** (recommended)

Then run the command.

---

## Which command should I use?

This depends on your system setup.

### Safest option (works most of the time)

```bash
python -m pip install package-name
```

Example:

```bash
python -m pip install pandas
```

Why this is better:

* Guarantees pip installs packages for **the Python you’re using**
* Avoids “installed but can’t import” errors

---

## Common beginner mistakes (very normal)

❌ Running pip commands inside Python:

```python
pip install numpy # ❌ wrong
```

❌ Running them in a `.py` file
❌ Forgetting to press Enter
❌ Using the wrong Python version (`python` vs `python3`)

---

## Quick sanity check

After installing a package:

```bash
python
```

Then inside Python:

```python
import numpy
```

If no error → you’re good 🎉

---

## TL;DR

* **Python** = the language
* **pip** = installs Python libraries
* **bash commands** run in the **terminal**
* Use:

 ```bash
 python -m pip install package
 ```

`NotImplementedError` is meant for:

* **stub / template code**
* **abstract methods**
* **functions you haven’t written yet**

Not for finished implementations.

✔ Reachable
✔ Clean
✔ What graders expect in starter code

If part of the logic is optional or not done yet:

```python
def suggest_slots(...):
```

Now it’s reachable **by design**.
