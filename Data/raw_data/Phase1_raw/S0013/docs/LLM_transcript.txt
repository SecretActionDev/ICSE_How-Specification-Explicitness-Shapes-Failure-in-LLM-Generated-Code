Prompt 2: "given a function, inputs are a resource map [str:int], request map [str:int] output a bool true false response the request map can request the same resource multiple times and returns false if the count of that request sums more than the resource"
LLM Response 2: "ure — here’s a clear Python function that does exactly what you described.

📌 What the function should do

Input:

resource_map: A dict of available resources, e.g. {"cpu": 4, "mem": 16}

request_map: A dict of resource requests, but it can have the same key multiple times (e.g., from different parts of a request)

Output:

True if the total requested value for each resource does not exceed what’s available

False if any resource is over-requested

✅ Python implementation
from collections import Counter

def is_allocation_feasible(resource_map: dict[str, int], request_map: dict[str, int]) -> bool:
 """
 Check if the total requests for each resource do not exceed the available resources.
 request_map may have repeated resource names, so we sum them first.
 """

 # Sum all requests per resource
 total_requests = Counter(request_map)

 # Check against available resources
 for res, req_count in total_requests.items():
 available = resource_map.get(res, 0)
 if req_count > available:
 return False

 return True"
